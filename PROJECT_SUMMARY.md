1. **Instala el add-on**:
   - Lee INSTALLATION.md para instrucciones paso a paso
   
2. **Inicializa los sistemas**:
   ```
   /function init_nbt_systems

3. **Obtén los items iniciales**:
   ```
   /function give_nbt_items

   - Añade más idiomas
